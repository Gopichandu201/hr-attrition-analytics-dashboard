# Hr-Attrition-Analytics-Dashboard
# HR Attrition Analysis

This project analyzes employee attrition using data analytics techniques to identify key factors influencing workforce turnover and support data-driven HR decisions.

## Tools Used

* SQL (MySQL)
* Power BI
* Data Cleaning
* Data Visualization
  
## Dataset

The dataset includes:

* Age, Department, Job Role
* Salary Level
* Monthly Working Hours
* Satisfaction Level
* Years at Company
* Promotion (Last 5 Years)
* Attrition Status


## Project Workflow

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* SQL Queries for Data Analysis
* Dashboard Creation using Power BI


## SQL Analysis

* Total Employees Count
* Attrition Count and Rate
* Department-wise Attrition
* Salary-wise Attrition
* Promotion Impact Analysis

## Key Insights

* Low salary employees have higher attrition
* Sales and Support departments show high attrition
* Low satisfaction leads to higher employee turnover
* Lack of promotion increases attrition risk

## Dashboard Features
Total Employees
Attrition Count
Attrition Rate
Average Satisfaction
Attrition by Department
Attrition by Salary

## Outcome

* Developed an interactive Power BI dashboard to monitor and analyze employee attrition
* Identified key factors influencing attrition such as salary level, job satisfaction, and promotions
* Enabled clear visualization of department-wise and salary-based attrition trends
* Provided actionable insights to support HR decision-making and improve employee retention strategies
* Strengthened skills in SQL, data analysis, and data visualization through real-world application
